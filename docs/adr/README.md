# Architecture Decision Records

ADRs record accepted architectural decisions for alignx.

## Current ADRs

- `adr-001-language-and-build.md` - language, build system, and dependency management
- `adr-002-axf-format-design.md` - AXF format direction
- `adr-003-index-design.md` - index design direction
- `adr-004-compression-and-acceleration-strategy.md` - CPU/GPU and lossless compression strategy

## Guidelines

- Add a new ADR when a decision changes project architecture, file format,
  dependency strategy, or benchmark methodology.
- Keep accepted decisions concise and stable.
- Put exploratory alternatives in `docs/research/` until a decision is made.
